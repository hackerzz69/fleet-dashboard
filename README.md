# 🚛 Fleet Intelligence Dashboard

A professional fleet-level IFTA and mileage dashboard designed for owner-operators and small fleet operations.

This dashboard provides executive-level visibility into quarterly mileage, fuel usage, IFTA state allocation, and trip reporting — without expensive fleet management software.

---

## 🔥 Features

- ✅ Quarter & Truck selection (multi-truck ready architecture)
- ✅ KPI summary cards
- ✅ IFTA mileage breakdown by state
- ✅ Recent trip reporting
- ✅ Fuel usage & MPG display
- ✅ Mobile responsive
- ✅ Clean SaaS-style UI
- ✅ Static site (no backend required)

## 🚀 Deployment (GitHub Pages)

1. Push repository to GitHub
2. Go to:
   ```
   Settings → Pages
   ```
3. Under **Build and Deployment**
   - Source: Deploy from branch
   - Branch: `main`
   - Folder: `/ (root)`
4. Save

Your dashboard will be live at:

```
https://yourusername.github.io/fleet-dashboard/
```
